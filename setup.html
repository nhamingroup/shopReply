Hướng dẫn cài đặt ShopReply

Phần mềm tự động trả lời tin nhắn cho shop bán hàng trên Facebook & Zalo.
Mọi dữ liệu đều được xử lý trên máy của bạn — không gì bị gửi lên cloud.

🔒 100% Local & Private ⚡ Cài đặt trong 5 phút 🚀 Tự động học càng dùng

Yêu cầu hệ thống

Hạng mục Yêu cầu tối thiểu
Hệ điều hànhWindows 10 trở lên (64-bit)
RAM8 GB (khuyến nghị 16 GB nếu dùng AI gợi ý)
Ổ cứng~2 GB trống (cho backend + model embedding)
Trình duyệtGoogle Chrome phiên bản 120+
MạngKhông cần internet để trả lời tự động. Cần internet để cài Chrome Extension

Tùy chọn: Cài Ollama (để có AI gợi ý)

Nếu bạn muốn ShopReply gợi ý câu trả lời bằng AI cho những câu hỏi mới:

  1. Tải Ollama tại: https://ollama.com/download
  2. Cài đặt bình thường (Next > Next > Install). Sau khi cài xong, cửa sổ chat Ollama sẽ tự mở — bạn có thể tắt cửa sổ này, Ollama vẫn chạy ngầm bình thường.
  3. Mở Command Prompt, chạy: ollama pull qwen2.5:7b
  4. Đợi tải xong (khoảng 4-5 GB)
ℹ️
Nếu không cài Ollama, ShopReply vẫn hoạt động bình thường — chỉ là không có AI gợi ý cho câu hỏi mới.
1

Cài Backend (1 lần duy nhất)

  1. ⬇ Tải ShopReply-Backend-v1.0.0.zip — giải nén ra sẽ có file ShopReply-Backend.exe
  2. Tạo 1 thư mục riêng, ví dụ: C:\ShopReply\
  3. Bỏ file .exe vào thư mục đó
  4. Double-click để chạy
  5. Lần đầu chạy sẽ mất 30-60 giây để tải model embedding (chỉ lần đầu)
  6. Biểu tượng ShopReply (hình tròn xanh chữ “S”) sẽ hiện ở khay hệ thống (góc dưới phải màn hình)
💡
Mẹo: Nếu không thấy biểu tượng, click mũi tên ^ ở khay hệ thống để hiện các biểu tượng ẩn.

Kiểm tra đã chạy thành công

  1. Click phải biểu tượng ShopReply ở khay hệ thống
  2. Chọn “Open Health Check”
  3. Trình duyệt sẽ mở và hiện trạng thái "status": "ok" — vậy là thành công!
2

Cài Chrome Extension

  1. Mở Chrome Web Store (link sẽ được cung cấp sau)
  2. Nhấn “Thêm vào Chrome” (Add to Chrome)
  3. Nhấn “Thêm tiện ích mở rộng” (Add extension)
  4. Ghim biểu tượng ShopReply vào thanh công cụ:
    • Click biểu tượng puzzle (tiện ích mở rộng) ở góc phải Chrome
    • Tìm ShopReply > click biểu tượng ghim
3

Import bộ câu hỏi - trả lời

Đây là bước quan trọng nhất — bạn cần “dạy” cho bot biết cách trả lời khách hàng.

Cách 1: Import từ file Khuyến nghị

  1. Chuẩn bị file CSV với 2 cột: question và answer

Ví dụ:

question,answer
"Giá áo hoodie?","Áo hoodie giá 350k, size S-XL ạ"
"Ship bao lâu?","Ship nội thành 1-2 ngày, ngoại thành 3-5 ngày ạ"
"Có COD không?","Có COD nha bạn, nhận hàng rồi trả tiền ạ"
  1. Click biểu tượng ShopReply trên Chrome > “Cài đặt”
  2. Chọn tab “Import”
  3. Kéo thả file CSV vào hoặc click để chọn file
  4. Xác nhận import

Cách 2: Thêm từng câu Tùy chọn

  1. Mở cài đặt ShopReply > tab “Q&A”
  2. Click “Thêm mới”
  3. Nhập câu hỏi và câu trả lời
  4. Lưu lại

Cách 3: Scan lịch sử chat cũ Cần Ollama

  1. Mở Facebook Messages hoặc Zalo Web trên Chrome
  2. Click biểu tượng ShopReply > “Scan lịch sử”
  3. Chọn những cuộc hội thoại là khách hàng (bỏ qua bạn bè, nhóm chat...)
  4. ShopReply sẽ tự động trích xuất các cặp câu hỏi - trả lời
  5. Review và xác nhận

Cách 4: Không cần import — bot tự học Tùy chọn

Nếu bạn không có data sẵn:

  1. Để bot chạy ở chế độ “Suggest only” (gợi ý, không tự động gửi)
  2. Mỗi khi khách hỏi → bạn tự viết câu trả lời → bot tự lưu vào database
  3. Lần sau gặp câu tương tự → bot tự động trả lời luôn
  4. Sau 1-2 tuần → bot đã học đủ để tự động 80-90% câu hỏi
4

Bắt đầu sử dụng

  1. Đảm bảo ShopReply Backend đang chạy (biểu tượng xanh ở khay hệ thống)
  2. Mở Facebook Messages (facebook.com/messages) hoặc Zalo Web (chat.zalo.me)
  3. Khi có tin nhắn khách hàng, ShopReply sẽ xử lý như sau:
✅
Độ chính xác ≥ 85%
ShopReply tự động trả lời luôn
💬
Độ chính xác 50-84%
Hiện bảng đề xuất, bạn chọn/sửa rồi gửi
✨
Câu hoàn toàn mới
Gợi ý từ AI (nếu có Ollama), bạn tự viết rồi gửi

Mỗi lần bạn trả lời câu mới → ShopReply tự động học, lần sau gặp câu tương tự sẽ tự trả lời.

🚀
Càng dùng nhiều, bot càng thông minh. Sau 1-2 tuần, bot có thể tự động trả lời 80-90% câu hỏi.

Khởi động cùng Windows (tùy chọn)

Để ShopReply tự động chạy mỗi khi bật máy:

  1. Click phải biểu tượng ShopReply ở khay hệ thống
  2. Chọn “Start with Windows” (sẽ hiện dấu check khi đã bật)
  3. Để tắt: click lại “Start with Windows” lần nữa
